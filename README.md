# 数学有什么用：从买菜到人工智能的数学之旅

一个用生活场景讲数学的科普系列，共 12 期。

---

## 系列简介

数学不是做题机器，也不是高考工具。它藏在你每天的生活里：买菜时的比较、装修时的测量、看天气预报时的概率……甚至手机导航、人脸识别、AI 聊天背后，都是数学在默默工作。

这个系列从最日常的场景出发，用最直白的语言，讲清楚数学到底有什么用。

- **生活场景切入**：每篇文章从一个真实的日常问题出发
