第二篇　初中数学：逻辑思维的启蒙

如果说小学数学给了孩子"计算"的能力，初中数学给的就是"推理"的能力。从方程到几何证明，初中数学的每个知识点都在训练一件事——逻辑思维。而逻辑思维，是一个人一辈子最值钱的免费礼物之一。

方程是初中数学的第一个"大BOSS"。很多孩子从此觉得数学变难了、学不懂了、人生没有意义了……但如果你告诉他：方程其实就是小学学过的天平，只不过天平的一个盘子里放了一个你不认识的东西——他可能就不那么害怕了。

6.1　方程就是带未知数的天平

小学里的天平：3 + 5 = 8，两边平衡。方程只是在天平上放了个"神秘重量"：x + 3 = 7，两边也平衡。如图6-1所示。

图6-1　方程就是带未知数的天平

x是你不认识的数，但你知道"它加上3等于7"。怎么找出x？天平两边同时减去3，就剩下x = 4了。这就是"移项"的本质：保持平衡，把认识的数全移到一边，让x孤零零地站在另一边，它就暴露无遗了。

更复杂的也一样。2x + 5 = 13：两边减5得到2x = 8；两边除以2得到x = 4。每一步都是"天平两边做同样的事"，从不偏心。

方程思维在生活里无处不在，只是你没意识到。"我需要再存多少钱才能凑够首付"——这就是个方程：现有存款 + x × 每月储蓄 = 首付金额。解x，知道还要攒几个月。

6.2　鸡兔同笼：古老的方程题

"鸡兔同笼"是大约1500年前《孙子算经》里的题目，堪称史上最长寿的数学题。题目是：笼子里鸡和兔头共35个、脚共94只，鸡兔各几只？如图6-2所示。

图6-2　鸡兔同笼问题

设鸡x只、兔y只，列方程组：

头：x + y = 35

脚：2x + 4y = 94

从第一个方程得x = 35 - y，代入第二个，解得y = 12，x = 23。答案：23只鸡、12只兔。

这道题1500年前就有人做，但它蕴含的"用方程组解多未知量问题"的思想，至今仍是数学和工程的核心方法。现代的线性规划、最优化，本质都是在解更大规模的方程组——只是未知数从2个变成了几千万个。

6.3　一元二次方程与抛物线

当方程里的x带上了"平方"，就变成了一元二次方程，比如x² - 2x - 3 = 0。如图6-3所示，它的图像是一条抛物线——就像你投篮时篮球的轨迹。

图6-3　一元二次方程与抛物线

抛物线和x轴的交点就是方程的解。图中两个交点x=-1和x=3就是这个方程的两个解。如果抛物线完全在x轴上方（和x轴不相交），方程就没有实数解——就好像你用一个大碗扣地板，碗和地板面没有交点。

抛物线在物理里极其重要：投出去的石头、射出去的炮弹、跳远运动员的轨迹，都走抛物线。卫星天线（锅形天线）的形状是抛物面——因为抛物面有个神奇性质：所有平行于轴的信号都汇聚到焦点。这就是为什么你家锅形天线能把卫星信号"接住"，而不是一锅粥乱成一团。

八卦一下：方程（Algebra）这个词来自公元820年波斯数学家花拉子米写的一本书，书名里有个词"al-jabr"（还原），演变成了Algebra。"算法"（Algorithm）这个词则直接来自他的名字。也就是说，今天所有程序员天天挂在嘴边的"算法"，名字来自1200年前一个波斯数学家。历史挺有趣的。

7.1　三条边确定一个三角形

初中几何有个看似平凡、实则深刻的结论：给定三角形的三条边长度，这个三角形的形状就完全确定了。如图7-1所示。

图7-1　三条边确定唯一的三角形

你拿三根棍子，长3厘米、4厘米、5厘米，不管怎么摆，摆出的三角形都一模一样。但如果只给两根棍子，可以摆出无数种不同的三角形。

这背后的思想是：条件充分，才能确定结论。这就好比破案——你只知道嫌疑人穿黑衣服，满大街都是；但你知道他穿黑衣服、戴红帽子、左手有纹身、身高一米八，条件足够多了，嫌疑人就唯一确定了。数学和侦探，道理相通。

7.2　全等三角形的判定

全等三角形是形状和大小完全相同的两个三角形。如图7-2所示，判定全等有三种经典方法：

图7-2　全等三角形的三种判定方法

——SSS（边边边）：三条边对应相等

——SAS（边角边）：两条边和它们的夹角对应相等

——ASA（角边角）：两个角和它们的夹边对应相等

在工业生产里，质检员需要知道"检测哪几个参数就能保证产品合格"——和SSS判定全等的思路完全一样：找最少、最可靠的条件，得出确定的结论。

7.3　从条件到结论的逻辑链

如图7-3所示，数学的本质就是：从已知条件出发，通过逻辑推理，得到确定的结论。

图7-3　数学的本质：从条件到结论

几何证明题不让你说"看起来像"，必须一步一步推，每步都有根据。这种"前提→推理→结论"的思维，以后写论文要用，做工程要用，甚至在生活中辨别谣言也要用。你说，学几何证明有什么用？用大了。

7.4　勾股定理：最实用的数学定理

直角三角形两条直角边的平方之和等于斜边的平方：a² + b² = c²。最著名的例子：3² + 4² = 5²（9+16=25）。如图7-4所示。

图7-4　勾股定理：a² + b² = c²

勾股定理的现实应用多得数不清：

——建筑工人用"三四五法"验证墙角是直角：量3米、4米，对角线恰好5米就直了

——电视机"55英寸"指的是对角线，宽高比16:9时用勾股定理算出实际宽高

——帐篷中心杆高1.5米、底边到中心2米，斜面布料至少要√(1.5²+2²)=2.5米

勾股定理已知的证明方法超过400种——数学史上没有哪个定理有这么多证明。其中最有趣的是美国第20任总统加菲尔德在1876年发现的一种，用梯形来证明。一位总统发表数学证明，历史上绝无仅有，也算是"兴趣爱好"的极限操作了。

给孩子的挑战：中国古代数学家赵爽用四个全等直角三角形和一个小正方形拼成大正方形，优美地证明了勾股定理——这个"弦图"今天还印在教材里。让孩子自己剪纸试试拼一拼，比抄100遍公式管用。

7.5　小结

初中几何看起来是在画三角形、做证明题，但它真正锻炼的是：条件要充分、推理要严密、结论要可靠。这套逻辑思维能力，无论以后做什么——搞技术、做研究、当律师、开公司——都是最核心的竞争力之一。而且，它是数学免费送的。
