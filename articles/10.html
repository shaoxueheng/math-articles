第三篇　高中数学：打开科技之门

高中数学开始真的变难了。很多家长看到孩子的作业，脑子里只会浮现三个字：看不懂。

但偏偏是这些让人头疼的内容——数列、双曲线、向量、排列组合——在现代科技里扮演着关键角色。飞机追踪、手机定位、密码学、人工智能，全都建立在这些基础之上。

学不会不要紧，但至少要让孩子知道"这玩意儿是有用的"。今天我们从天上飞的飞机说起。

11.1　等差数列与等比数列

数列就是一列按规律排列的数。如图11-1所示，高中数学里最重要的两种：

图11-1　等差数列与等比数列的对比

——等差数列：每次加相同的数。2, 5, 8, 11, 14...公差3。像走楼梯，每步一样高。

——等比数列：每次乘相同的数。1, 2, 4, 8, 16...公比2。像滚雪球，越滚越猛。

等差数列很温和，等比数列很暴力。传说国际象棋发明者让国王在棋盘上放麦粒：第一格1粒、第二格2粒、第三格4粒……每格翻倍。国王觉得很少，国王很快就后悔了——64格总共需要2⁶⁴-1 ≈ 1.8×10¹⁹粒麦子，相当于全世界几百年的产量。这就是等比数列：初期骗你，后期揍你。

复利增长、病毒传播、细胞分裂、AI模型参数量的增长……凡是"翻倍增长"的，都是等比数列。认识它，你就不会被"每天进步1%"的鸡汤感动，而是真的去算一算。（答案：1.01³⁶⁵ ≈ 37.8，确实厉害。）

11.2　插值：填补数据的空白

现实世界的数据往往不完整。气象站每小时测一次温度，但你想知道某半小时的温度？如图11-2所示。

图11-2　插值：用已知数据推算未知数据

最简单的"线性插值"：8点15度、9点19度，8点半大约是15+(19-15)×0.5=17度。更复杂的插值能画出光滑曲线，让估算更精准。

插值在很多地方都在默默工作：气象预报、医学影像（CT扫描层间的图像）、视频播放（补帧让画面更流畅）。

你早就在用插值："去年70公斤、今年74公斤，有人问你半年前多重，你说大概72公斤吧"——恭喜，你的大脑在做线性插值，而且速度比计算机快。

11.3　卡尔曼滤波：从嘈杂数据中找到真相

测量数据往往有误差、有噪声，怎么提取真实信息？如图11-3所示。

图11-3　卡尔曼滤波效果

卡尔曼滤波的核心思想两步循环：第一步预测（根据上一时刻预测这一时刻），第二步修正（用实际测量修正预测）。就像你开车上班：你知道平时30分钟，导航说45分钟，你会折中说"大概35分钟"。你的大脑在做卡尔曼滤波，只是你不知道而已。

1960年，卡尔曼发表了这个算法。几年后，NASA把它用在了阿波罗登月飞船上。今天，手机GPS定位、飞机导航、自动驾驶汽车的传感器融合，都在使用卡尔曼滤波的变种。一个算法，改变了人类探索宇宙和出行的方式。

11.4　ADS-B与雷达标校

天上每一架民航飞机都在持续广播ADS-B位置信号。如图11-4所示。

图11-4　ADS-B与雷达标校

ADS-B来自GPS，比较准确。地面雷达也在追踪飞机，但有各种系统误差。"雷达标校"就是用精确的ADS-B数据修正雷达的偏差。这个过程用到：插值（对齐不同频率的数据），卡尔曼滤波（去除噪声），统计分析（计算系统偏差）。高中学的数列、插值，在追踪天上飞机的系统里发挥着实实在在的作用。

11.4.1　从雷达到自动驾驶

雷达标校的技术不止用于航空管制。自动驾驶汽车有激光雷达、毫米波雷达、摄像头、超声波多种传感器，每种都有自己的误差特性。要安全行驶，就要把所有传感器数据融合起来，取长补短。这个"传感器融合"的核心算法，正是卡尔曼滤波。没有高中数列和滤波的基础，就没有自动驾驶。

11.4.2　音乐里的等比数列

你可能不知道，音乐和等比数列有深厚的渊源。钢琴键盘上，相邻两个半音的频率之比固定，都是2的1/12次方（约1.0595）——所有半音的频率构成等比数列！

中央C频率261.6Hz，高八度的C是523.2Hz，正好是两倍。12个半音各乘以1.0595，走完一圈正好翻倍。这叫"十二平均律"。巴赫的《十二平均律钢琴曲集》就是为了展示这种数学美感而创作的。数学和音乐，比你想象的要亲密得多。

11.5　小结

数列帮你理解"有规律变化的量"，插值帮你"从已有数据推算未知数据"，卡尔曼滤波帮你"从不完美数据里找到真相"。这套工具组合，构成了现代数据处理的基础设施。高中数学看起来枯燥，但它是现代科技的语言——学不好没关系，但让孩子知道它在哪里发挥作用，就值了。

12.1　双曲线的定义

高中圆锥曲线里，双曲线的定义乍看抽象，但它的数学结构完美对应一个现实场景——"如果知道你到两个固定点的距离之差，你的位置就被约束在一条特定曲线上"。这正是飞机定位的数学基础。

【双曲线定义】

平面内，到两个定点 F₁、F₂（焦点）的距离之差的绝对值等于常数的点的轨迹，叫做双曲线。设两距离分别为 d₁、d₂，则：

｜d₁ − d₂｜ = 2a（常数，a > 0）

图12-1　双曲线上任意点 P 到两焦点的距离之差为常数。P 在右支，d₁（到 F₁）远大于 d₂（到 F₂），但 d₁ − d₂ 始终等于 2a。

如图12-1所示，P 是双曲线右支上一点，F₁ 是左焦点，F₂ 是右焦点。P 距 F₂ 很近（d₂ 短），距 F₁ 很远（d₁ 长），但两者之差 d₁ − d₂ 等于固定值 2a。无论 P 怎么沿右支移动，这个差值永不改变。

双曲线有两支：右支上的点离 F₂ 更近（d₁ − d₂ = +2a）；左支上的点离 F₁ 更近（d₂ − d₁ = +2a）。合写就是 ｜d₁ − d₂｜ = 2a。这个性质看似抽象，下一节就能看到它如何直接"描述"飞机的位置。

12.2　时差定位：从时间差推出双曲线

地面有两个接收站 A 和 B，飞机不断广播无线电信号（ADS-B 或应答机脉冲），A 和 B 各自记录信号到达的精确时刻。推导分三步。

第一步：时间差 ↔ 距离差

无线电信号以光速 c ≈ 3×10⁸ m/s 传播。设飞机到 A 站的距离为 d_A，到 B 站的距离为 d_B，信号分别在 t_A 和 t_B 时刻到达两站。因为飞机离 A 近（d_A < d_B），所以 A 先收到信号（t_A < t_B）。两站之间的时间差 Δt 对应距离差：

d_B − d_A = c × Δt

时间差（秒）× 光速（米/秒）= 距离差（米）。接收站只要精确记录 Δt，就能算出 d_B − d_A。

图12-2　飞机离 A 站更近，信号先到 A（t_A），后到 B（t_B）。时间差 Δt 乘以光速即得距离差 d_B − d_A。

第二步：距离差 = 常数 → 双曲线！

关键在于：测出 Δt 后，d_B − d_A = c × Δt 是一个固定数值。再看双曲线的定义——"到两定点的距离差为常数"。两句话完全一致！

因此，飞机必然位于以 A、B 为焦点、距离差等于 c×Δt 的那条双曲线的某一支上。由于 A 先收到信号（飞机离 A 更近，d_A < d_B），飞机在靠近 A 的那支上。

仅凭 A、B 两站的时间差，飞机的可能位置就从"整个天空"收缩到了"一条曲线"——这已经非常强大了。

12.3　三站定位：两条双曲线的交点

一条双曲线上仍有无数点，还需要第二个约束。引入第三个接收站 C，记录飞机信号到达 A 和 C 的时间差 Δt'，同样的方法得到第二条双曲线（以 A、C 为焦点，差值为 c×Δt'）。

飞机必须同时满足两个距离差约束，因此只能在两条双曲线的交点处。通常两支曲线在有效飞行区域内只有一个交点——这就是飞机的精确位置。

图12-3　三个地面接收站 A、B、C 产生两条双曲线（蓝线由 A-B 时差确定，橙线由 A-C 时差确定），两线交点即为飞机精确位置。

如图12-3所示，蓝色曲线和橙色曲线在飞机位置处相交。A 站同时参与两个站对，所以也叫"参考站"。实际系统中通常布置 4～6 个接收站，多余的方程用于提高精度和排除虚假交点。这套技术称为多点定位（Multilateration，MLAT），是目前最主要的广域飞机位置监视手段之一。

与 GPS 对比：GPS 用"到卫星的距离"画球面，多个球面相交得出位置；MLAT 用"到地面站的距离差"画双曲线，多条曲线相交得出位置。两者原理高度相似，都是高中几何在现代工程中的直接应用。

12.4　小结

双曲线不只是考试里让人头疼的曲线，它是航空管制系统、GPS 导航、手机定位的数学核心。高中时觉得无聊的圆锥曲线，其实是追踪每一架飞机的秘密武器。从"到两定点距离差为常数"到精确定位一架飞机，中间只差一步推导：时间差 × 光速 = 距离差 = 双曲线参数。
